# Full-Stack Authentication System (Task 1)

## Project Overview
A fully integrated web application featuring user authentication and cloud database management. 

## Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** Aiven MySQL (Managed Cloud)
* **Hosting:** Railway
* **Security:** bcryptjs (password hashing), JWT (JSON Web Tokens)

## Deployment
* **API Base URL:** https://codveda-fullstack-tasks-production.up.railway.app
* **Database:** Hosted via Aiven Cloud with SSL encryption.

## Local Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file based on `.env.example`.
4. Run locally: `npm start`.
